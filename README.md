# DctWatermarkApp

Приложение на C# (Windows Forms) для встраивания и извлечения цифровых водяных знаков
с использованием дискретного косинусного преобразования (DCT).

## Возможности

- Загрузка изображения
- Встраивание текстового водяного знака
- Извлечение встроенного водяного знака
- Сохранение результата
- Расчет метрик качества изображения:
  - MSE
  - PSNR
  - SSIM

## Используемые технологии

- C#
- .NET Framework 4.8
- Windows Forms
- DCT (Discrete Cosine Transform)

## Структура проекта

- Program.cs — точка входа
- MainForm.cs — логика пользовательского интерфейса
- MainForm.Designer.cs — описание интерфейса
- WatermarkLogic.cs — алгоритмы встраивания и извлечения
- DctHelper.cs — реализация DCT/IDCT
- ImageMetrics.cs — вычисление MSE, PSNR и SSIM

## Запуск

1. Откройте файл решения (.sln) или файл проекта (.csproj) в Visual Studio.
2. Соберите проект.
3. Запустите приложение.

## Примечание

Папки `bin`, `obj` и `.vs` намеренно не включаются в репозиторий, так как
они автоматически создаются средой разработки.
